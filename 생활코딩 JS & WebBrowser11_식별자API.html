식별자 API

HTML에서 Element의 이름, id, class는 식별자로 사용된다. 식별자 API는 이 식별자를 가져오고 변경하는 역할을 한다.

tagName : 선택자로 선택한 태그의 이름을 정의해준다

태그의 이름을 정의(읽기전용)해주는것만 가능하고, 변경은 불가능하다

id를 바꿀 수 있긴 한데, 스크립트에서는 이전id로 선택되어있기 때문에, 헷갈려서 안쓸듯

Element.className

태그에 class를 한번에 여러개 등록 가능하지만, 수정하거나 추가, 삭제 할 때에 다 써줘야 하므로 매우 불편하다.
  • HTML
  • CSS
  • JavaScript

Element.classList

className의 불편함을 보완했다.

DOMTokenList : 특정 태그 선택자의 classList를 console로 찍었을 때, DOMTokenList라는 유사배열형태가 나온다. 해당 태그의 class리스트들도 있으며, __proto__에 부모객체의 메소드들도 상속받아져 있다.

선택자.classList[num] : num번째 class
선택자.classList.length : class의 갯수
선택자.classList.add('새 class') : class 추가
선택자.classList.remove('기존 class') : class 제거
선택자.classList.toggle('class') : 해당 class가 없으면 추가하고 있으면 제거하고
