속성은 HTML에서 태그명만으로 부족한 부가적인 정보

  • 선택자.getAttribute('속성명') : 해당 속성정보 갖고오기
  • 선택자.setAttribute('속성명', '변경 내용') : 해당 속성정보 변경하기
  • 선택자.removeAttribute('속성명') : 해당 속성 삭제
  • 선택자.hasAttribute('속성명') : 해당 속성 존재 여부 파악

속성과 프로퍼티

HTML에서의 Element속성을 JavaScript를 통해 제어가 가능하다
  • 속성방식 : setAttribute등으로 제어하는 방식
  • 프로퍼티 방식 : className, classList같은것들
프로퍼티 방식이 좀 더 간편하고 빠르지만, JavaScript에서 이미 이름이 정해진 경우때문에 종종 HTML의 이름과 다른 경우가 있다.(class => className, rowspan => rowSpan)
