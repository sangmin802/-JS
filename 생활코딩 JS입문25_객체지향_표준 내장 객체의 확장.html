Standard Built-in Object(내장객체) : 자바스크립트가 기본적으로 제공하는 객체

  • Object : 객체
  • Function : 함수 혹은 메소드
  • Array : 배열
  • String : 문자열 변환
  • Boolean : true & false
  • Number : 숫자열 변환
  • Math : 숫자 함수
  • Date : 날짜 함수
  • RegExp : 정규표현식

내장객체에 기능 추가하기

prototype을 위의 Array를 활용한 함수에 접해보기

해설

1. 배운바를 통해 new Array()로 배열을 생성하는것 또한, 객체생성자임을 알 수 있다.
2. 그렇다면 내장객체 생성자 또한 고유의 원형객체인 prototype을 모두 갖고있다.
3. 우리는 이러한 고유의 원형객체인 prototpye에 새로운 메소드나 프로퍼티를 추가하여 해당 생성자를 통해 만들어지는 모든 결과물에, 내가 prototype에 넣은 메소드나 프로퍼티를 사용할 수 있다.
4. new Array()라는 배열 객체 생성자의 prototype에 random이라는 메소드를 넣는다
1) Array.prototype.메소드 이기 때문에, 해당 메소드의 이름이 많이 짧아질 수 있다. getRandomValueFromArray -> random
2) 해당 메소드안의 매개변수는 this가 된다(객체생성자 내의 this는 해당 객체가 된다. 여기서는 arr자체가 this가 됨.)
3) 호출할 때, arr.random()
var arr = new Array('상민', '지찬', '상혁', '태균', '원빈'); 여기서 var arr는 new Array의 prototype또한 사용할 수 있게 됨.
